### About Qrier

Qrier fetches and sends email. In Ruby. It is part of the [Build an Email Application With Ruby Tuts+ Course](https://code.tutsplus.com/courses/build-an-email-application-with-ruby).

#### Installation

Install it via:

    $ gem install qrier

#### Usage

Run `qrier` in your terminal:

    $ qrier
